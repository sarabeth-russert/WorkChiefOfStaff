# Page Header Illustrations

This directory contains header/hero illustrations for the main feature pages.

## Required Files

Place your generated page header images here:

- `expedition-header.png` - Expedition page (agent orchestration)
- `trading-post-header.png` - Trading Post page (PM2 management)
- `map-room-header.png` - Map Room page (knowledge base)
- `outpost-header.png` - Outpost page (terminal & git tools)

## Image Specifications

- **Format:** PNG (transparent or cream background)
- **Size:** 1920x600px (landscape banner) or 1200x400px (balanced)
- **Style:** Vintage SEA aesthetic matching character illustrations
- **Color Palette:** Warm earth tones per page theme

## Generation Prompts

See `/PAGE_ILLUSTRATIONS_GUIDE.md` in the root directory for detailed AI image generation prompts.

## Usage

These images will be displayed as:
- Hero banners at the top of each page
- Background illustrations (optional)
- Navigation card previews (optional)
