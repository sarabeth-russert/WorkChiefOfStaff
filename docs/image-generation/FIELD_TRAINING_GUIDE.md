# Field Training Images - AI Generation Guide

## Overview

Generate all vintage-style illustrations for the Field Training page — a page header, dashboard icon, and eleven cognitive domain icons. All images share the Adventureland SEA aesthetic used across the rest of the application.

**Style:** Vintage 1950s-60s Disney Adventureland attraction poster, SEA (Society of Explorers and Adventurers) aesthetic, clean lines, warm lighting

---

## Image Specifications

- **Format**: PNG with transparent background
- **Header Size**: 1920x600px (landscape banner)
- **Icon Size**: 512x512px (square)
- **Style**: Consistent vintage poster aesthetic — flat illustration, hand-drawn, retro poster art
- **Color Palette**: Warm earth tones
  - Sand: `#E8D4A8`
  - Terracotta: `#D4735E`
  - Jungle Green: `#4A7859`
  - Sunset Orange: `#E87144`
  - Teal: `#479B99`
  - Mustard: `#DAA520`
  - Cream: `#FFF8E7`

## Style Consistency Tips

### For All 7 Images:
1. **Use the same AI service** for all images in one session
2. **Generate domain icons together** for maximum visual consistency
3. **Keep the SEA style consistent** — every prompt includes the Adventureland/SEA keywords
4. **Use similar lighting** — warm golden hour, lantern glow
5. **Maintain visual hierarchy** — foreground objects in focus, decorative elements supporting

### Style Keywords (present in all prompts):
- "Vintage 1950s-60s"
- "SEA (Society of Explorers and Adventurers)"
- "Vintage Disney Adventureland poster style"
- "Flat illustration, hand-drawn aesthetic, retro poster art"
- "Warm earth tones"
- "Square format 512x512px with transparent background"

---

## Generation Tips

1. **Try both prompts**: The detailed version gives richer results; the simpler version is more predictable
2. **Iterate**: Generate 3-4 variations of each and pick the best
3. **Transparent backgrounds**: Verify PNGs have true transparency, not cream halos
4. **Test at display size**: Domain icons display at ~40px, dashboard icon at ~80px — make sure they read clearly when small

### Recommended AI Tools:
- **DALL-E 3 (ChatGPT)**: Best for controlled compositions matching existing icons
- **Midjourney**: Excellent for vintage poster aesthetics
- **Stable Diffusion**: More iterations, good for refinement

---

## After Generation

1. **Save to directories**: Place PNGs in the paths listed in File Structure above
2. **Verify dimensions**: Header at 1920x600, icons at 512x512
3. **Verify transparency**: Icons should have transparent backgrounds
4. **Test in UI**: The `DomainIcon` component has emoji fallbacks already in place — images will display automatically once added
5. **Optimize** (optional): Use TinyPNG or Squoosh to reduce file sizes while maintaining quality
