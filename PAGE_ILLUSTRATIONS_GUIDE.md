## üîß Post-Generation Processing

### Optimize Images

Images at 1920x600 may be large. Optimize them:

```bash
# Using ImageMagick (if installed)
convert expedition-header.png -quality 85 -strip expedition-header.png

# Or use online tools:
# https://tinypng.com - Best compression
# https://squoosh.app - More control
```

**Target sizes:**
- Full quality: 200-500KB per image
- Optimized: 100-200KB per image

### Add Texture Overlay (Optional)

For extra vintage feel:

```bash
# Add paper texture overlay using ImageMagick
convert expedition-header.png \
  -colorspace gray \
  +level-colors '#3A3226','#FFF8E7' \
  -modulate 100,80 \
  expedition-header-vintage.png
```

## üìä Size Recommendations

### Performance vs Quality Trade-offs:

| Size | Use Case | Pros | Cons |
|------|----------|------|------|
| **1920x600px** | Desktop hero banners | High quality, immersive | 500KB-2MB per image |
| **1200x400px** | Balanced | Good quality, reasonable size | May pixelate on large screens |
| **960x300px** | Mobile-optimized | Fast loading, small size | Lower detail |

**Recommendation:** Generate at 1920x600, then create optimized versions:

```bash
# Create multiple sizes
convert expedition-header.png -resize 1920x600 expedition-header-full.png
convert expedition-header.png -resize 1200x400 expedition-header-medium.png
convert expedition-header.png -resize 960x300 expedition-header-small.png
```

Then use responsive images:

```jsx
<img
  src="/images/pages/expedition-header-medium.png"
  srcSet="/images/pages/expedition-header-small.png 960w,
          /images/pages/expedition-header-medium.png 1200w,
          /images/pages/expedition-header-full.png 1920w"
  alt="Expedition"
/>
```
